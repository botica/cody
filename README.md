<p>cody is a command-line agent using openrouter models for inference.</p>
<p>reasoning models output reasoning to console (in pink)</p>
<p>enter '!' instead of y/n to auto-confirm all tool calls for the turn.</p>
<p>use """ or ''' for multi-line input. close with delimiter on its own line or at end of a line.</p>
<p>ctrl-c once interrupts current run, twice exits the program.</p>
<p>> python agent.py</p>
<p>uses <a href="https://github.com/BurntSushi/ripgrep">ripgrep</a> for search</p>
